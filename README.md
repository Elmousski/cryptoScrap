### Comment ca marche ? 

1. Clone le repository

2. Install all the gems
```sh
$ bundle install
```
rails  db:create

3. Migrate db files
```sh
$ rails db:migrate
```
4. Execute seed file

5. Visit the local version

$ rails server
